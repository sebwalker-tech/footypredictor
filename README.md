# Championship Predictions

A private EFL Championship prediction league app.

Features include:
- Email/password-style local accounts
- Leaderboard with overall and gameweek views
- Prediction history and fixture results
- Admin management screens
- Imported 2025/26 spreadsheet history
- football-data.org API sync for Championship scores and table data

The API key is intentionally not committed. Add it in the Admin Panel after opening the app.
