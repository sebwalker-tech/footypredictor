# Championship Prediction App Design System

Build the app using a simple iPhone-first block-based design system.

## Style
The app should feel clean, sporty, premium and minimal. Use large rounded cards, oversized numbers, simple layouts and strong whitespace.

Do not use neon, heavy gradients, tiny tables, cluttered dashboards or realistic football imagery.

## Colours
- Background: #F4F4F1
- Surface: #FFFFFF
- Muted surface: #E8E8E2
- Text: #111111
- Muted text: #6B6B6B
- Border: #D9D9D2
- Primary green: #25C26E
- Warning: #F4C542
- Danger: #E64B3C

## Typography
Use one clean sans-serif font.
Use oversized numbers as a key visual feature.

## Layout Rules
- iPhone-first
- Safe area padding
- 20px horizontal padding
- 12px gap between cards
- 24px card border radius
- Full-width pill buttons
- Bottom navigation fixed to bottom

## Core Components
Create reusable components:
- AppShell
- PageHeader
- BlockCard
- StatBlock
- FixtureBlock
- PredictionInput
- PrimaryButton
- BottomNav

## Main Screens
Create these screens:
1. Home
2. Predictions
3. Fixture Detail
4. Leaderboard
5. Results
6. Leagues
7. Profile
8. Settings

## Navigation
Bottom tabs:
Home, Predictions, Leaderboard, Leagues, Profile.

## Design Priority
Every screen should be understandable in 3 seconds.
Each screen should use a small number of large blocks rather than lots of small UI elements.
